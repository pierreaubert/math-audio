ðŸ“Š About This Suite

This visualization suite compares BEM numerical solutions with exact analytical solutions for acoustic scattering problems. Tests are performed in 1D, 2D, and 3D to validate correctness across different geometries and frequency regimes.

1D Wave Propagation

Plane waves, standing waves, and damped propagation in one dimension.

Tests:

  • Plane wave: p(x) = exp(ikx)
  • Standing wave with nodes
  • Exponential damping
  • Convergence studies
1D

2D Cylinder Scattering

Rigid cylinder scattering using Bessel/Hankel function series.

Tests:

  • Rayleigh scattering (ka << 1)
  • Mie regime (ka ~ 1)
  • High frequency (ka >> 1)
  • Directivity patterns
  • Scattering cross-section
2D

3D Sphere Scattering

Rigid sphere scattering using Mie theory (spherical harmonics).

Tests:

  • Rayleigh regime (ka < 0.3)
  • Mie resonances (ka ~ 1)
  • Geometric optics (ka > 10)
  • Radar cross-section (RCS)
  • Surface pressure distributions
3D
